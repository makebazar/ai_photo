# Тест реферальной системы

## Бот
**@ai_photo_testast_bot**

## Переменные окружения (на сервере)

```bash
TELEGRAM_BOT_NAME=ai_photo_testast_bot
TELEGRAM_PARTNER_BOT_NAME=ai_photo_testast_bot
```

> **Важно:** писать **без @** в начале!

## Быстрый тест (вручную)

### Шаг 1: Зарегистрироваться как партнёр

1. Откройте @ai_photo_testast_bot в Telegram
2. Нажмите `/start`
3. Откройте интерфейс партнёра

### Шаг 2: Создать реферальную ссылку

1. В интерфейсе партнёра откройте раздел **"Реферальные ссылки"**
2. Нажмите **"Создать ссылку"**
3. Выберите тип: **"Для клиентов"**
4. Название: "Тест"
5. Нажмите **"Создать"**

### Шаг 3: Скопировать и отправить ссылку

1. Нажмите **"Копировать"** на созданной ссылке
2. Ссылка должна выглядеть так:
   ```
   https://t.me/ai_photo_testast_bot?start=cl_XXXXX
   ```
   или в формате с @:
   ```
   @ai_photo_testast_bot?start=cl_XXXXX
   ```
3. Отправьте ссылку в другой Telegram (второй аккаунт)

### Шаг 4: Зарегистрироваться по ссылке

1. Откройте ссылку со второго аккаунта
2. Нажмите **"Запустить"** бота
3. `/start` — автоматически применится реферальный код

### Шаг 5: Проверить привязку

**В админке** (https://www.aitest.mydashadmin.ru/admin):

1. Раздел **"Партнёры"** — найдите первого партнёра
2. Нажмите на него — должна быть статистика:
   - **Прямых клиентов**: 1
   - **Баланс**: должен вырасти после оплаты заказа

**Или через API**:
```bash
curl https://www.aitest.mydashadmin.ru/api/partner/{PUBLIC_ID}/clients
```

## Что тестируется

1. **Регистрация партнёров** (3 уровня):
   - Root партнёр (без приглашения)
   - L1 партнёр (приглашён Root)
   - L2 партнёр (приглашён L1)

2. **Создание заказов**:
   - Клиент 1 регистрируется через L1
   - Клиент 2 регистрируется через L2

3. **Оплата заказов**:
   - Заказ 1: 2990₽ (Standard)
   - Заказ 2: 4990₽ (Pro)

4. **Начисление комиссий**:
   - L2 получает 40% от своего клиента
   - L1 получает 10% от своего клиента + 5% от L2
   - Root получает 5% от L2

## Ожидаемые результаты

| Партнёр | Прямых клиентов | L1 | L2 | Баланс (₽) |
|---------|-----------------|----|----|------------|
| Root    | 0               | 1  | 1  | ~249       |
| L1      | 1               | 1  | 0  | ~299 + ~249 = ~548 |
| L2      | 1               | 0  | 0  | ~1996      |

## Проверка в админке

1. Откройте https://www.aitest.mydashadmin.ru/admin
2. Раздел "Партнёры" — должно быть 3 партнёра
3. Раздел "Заказы" — должно быть 2 заказа
4. Проверьте балансы партнёров

## Возможные ошибки

- **401 Unauthorized** — проверьте, что заголовок `X-Admin-Auth: 1` отправляется
- **Уже существует** — пользователи уже созданы, тест идемпотентный
- **Нет баланса** — проверьте логи сервера на предмет ошибок `allocateCommissions`
