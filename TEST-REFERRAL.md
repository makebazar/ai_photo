# Тест реферальной системы

## Запуск

```bash
# На сервере или локально (если сервер запущен)
cd /app
API_BASE=https://www.aitest.mydashadmin.ru node test-referral.js
```

## Что тестируется

1. **Регистрация партнёров** (3 уровня):
   - Root партнёр (без приглашения)
   - L1 партнёр (приглашён Root)
   - L2 партнёр (приглашён L1)

2. **Создание заказов**:
   - Клиент 1 регистрируется через L1
   - Клиент 2 регистрируется через L2

3. **Оплата заказов**:
   - Заказ 1: 2990₽ (Standard)
   - Заказ 2: 4990₽ (Pro)

4. **Начисление комиссий**:
   - L2 получает 40% от своего клиента
   - L1 получает 10% от своего клиента + 5% от L2
   - Root получает 5% от L2

## Ожидаемые результаты

| Партнёр | Прямых клиентов | L1 | L2 | Баланс (₽) |
|---------|-----------------|----|----|------------|
| Root    | 0               | 1  | 1  | ~249       |
| L1      | 1               | 1  | 0  | ~299 + ~249 = ~548 |
| L2      | 1               | 0  | 0  | ~1996      |

## Проверка в админке

1. Откройте https://www.aitest.mydashadmin.ru/admin
2. Раздел "Партнёры" — должно быть 3 партнёра
3. Раздел "Заказы" — должно быть 2 заказа
4. Проверьте балансы партнёров

## Возможные ошибки

- **401 Unauthorized** — проверьте, что заголовок `X-Admin-Auth: 1` отправляется
- **Уже существует** — пользователи уже созданы, тест идемпотентный
- **Нет баланса** — проверьте логи сервера на предмет ошибок `allocateCommissions`
